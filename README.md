## #FCC624 top : 실시간 시스템 모니터링
리눅스에서 시스템의 실시간 프로세스 정보를 확인할 수 있는 도구.

CPU, 메모리 사용량, 프로세스 정보 등을 실시간으로 모니터링할 수 있어 시스템 관리자에게 유용하다.
